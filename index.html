About the Grid

Wednesday's technical team currently recommend the use of Bootstrap's grid system due to its wide adoption, mobile-first responsive approach and rich documentation.

Introduction

Grid systems are used for creating page layouts through a series of rows and columns that house the content.

As CSS grid systems were devised before the advent of responsive design, the execution of a grid system has to be reconsidered with smaller screens and a mobile-first approach in mind.

Grid Types

  • Fixed Container - Extra small definitions only
  • Fixed Container - Extra small and small definitions only
  • Fluid Container - Extra small definitions only

Notice how the .col-xs-1 instances are broken at small browser widths such as those of a mobile phone. This is due to the 15px padding on either side adding up to a number greater than the intended width of 8.3% of the browser width, e.g. 26.6px, which is a column width when a 320px browser width is divided into 12 columns. This issue makes a 12 column grid on mobile impossible, and besides the column gutter widths would obviously be too small anyway. The solution is to limit ourselves to an 8, 6, 4 or 3 column grid for small browser widths.
